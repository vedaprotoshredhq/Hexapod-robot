# Hexap# 
Welcome to the development repository for my full hexapod robot project! I am starting by designing and drafting the physical mechanics for the legs before moving on to the chassis body.

## 📐 Engineering Blueprints
The physical dimension drawings and blueprints for the multi-jointed hexapod legs are uploaded right here in this repository.

### 🚀 Detailed Engineering Roadmap:
- [ ] 1. Draft physical dimension sketches and mechanical blueprint layouts.
- [ ] 2. Create detailed, hour-long 3D CAD design videos modeling the joints.
- [ ] 3. 3D print the individual mechanical structural components.
- [ ] 4. Physically assemble and verify the articulation of a single leg.
- [ ] 5. Wire up the leg electronics to test servo motor angles and load capacities.
- [ ] 6. Fabricate the central chassis body and assemble the entire 6-legged robot frame.
- [ ] 7. Integrate the core electronics, write the inverse kinematics code, and get the hexapod walking!
